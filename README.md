# Korea-Question
이어드림 4기 2차 모의 경진 대회입니다.
---

## 1. 소개
- LLM을 활용하여 Mulitple - Choice Question (다지선다문제)를 해결하는 대회입니다.

---

## 2. 데이터
- Input : 문제 (Question). 선택지 (Multiple - Choice)
- Output : 문제별 정답 선택지 (0 ~ 3)

---

## 3. 주의 사항
- Participant Rules
  - 외부 데이터 사용 가능
- Submission Rules
  - 제출 횟수 제한 - 팀별 하루 5회
  - Submission.csv 파일 제출 -> 리더보드

---

## 4.  평가 지표
- 평가 지표 : Accuracy
- 전체 질문 중 정답을 맞힌 질문의 비율 (만점 1.0)
